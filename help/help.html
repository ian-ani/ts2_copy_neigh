HELP

This software has been created to be used with The Sims 2 to be able to have a copy of a neighborhood of your choice. It is recommended to read this Help file fully before using this software and to have a backup of your directory of choice.

SOURCE PATH

INSTALLATION NEIGHBORHOODS

If you want to copy a clean neighborhood (non-played neighborhoods) from base game, the available directories are the following:

  • N001 corresponds to Pleasantview
  • N002 corresponds to Strangetown
  • N003 corresponds to Veronaville
  • Tutorial

Its path will be something similar to this one: \The Sims 2 Ultimate Collection\Double Deluxe\Base\TSData\Res\UserData\Neighborhoods\[neighborhood code]

If you want to copy clean neighborhoods from an expansion pack:

  • G001 corresponds to Riverblossom Hills (The Sims 2: Seasons)
  • F001 corresponds to Desiderata Valley (The Sims 2: FreeTime)
  • E001 corresponds to Belladonna Cove (The Sims 2: Apartment Life)

Its path will be something similar to these:

  • G001: \The Sims 2 Ultimate Collection\Seasons\TSData\Res\UserData\Neighborhoods\G001
  • F001: \The Sims 2 Ultimate Collection\Free Time\TSData\Res\UserData\Neighborhoods\F001
  • E001: \The Sims 2 Ultimate Collection\Apartment Life\TSData\Res\UserData\Neighborhoods\E001

PLAYED NEIGHBORHOODS

If what you want is to copy a played neighborhood or a custom neighborhood, its default path is the following:

[Drive]:\Users\[your username]\Documents\EA Games\The Sims™ 2 Ultimate Collection\Neighborhoods\[neighborhood code]

DESTINATION PATH

What I recommend for easily incrementing the code number from the folder is that the destination path should be as follows:

[Drive]:\Users\[your username]\Documents\EA Games\The Sims™ 2 Ultimate Collection\Neighborhoods

EXAMPLE

If you want to copy Pleasantview (neighborhood with code N001) its source path will be: \The Sims 2 Ultimate Collection\Double Deluxe\Base\TSData\Res\UserData\Neighborhoods\N001

But once you have to choose its destination path you MUST NOT choose a neighborhood code: [Drive]:\Users\[your username]\Documents\EA Games\The Sims™ 2 Ultimate Collection\Neighborhoods
